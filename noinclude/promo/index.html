Reader's Web

A better web for reading

Reader's Web is a part of the World Wide Web where the reader, not the publisher, decides what web pages look like. From the reader's perspective all Reader's Web pages look the same regardless of what website they come from. And the reader can change the theme for all of them at once. Advantages of Reader's Web pages: The publisher doesn't have to worry about selecting a theme for their website. The reader can read comfortably using one theme across many websites. New types of connections are possible like visible connections between parts of texts on two documents. You're currently viewing a page that is a part of the Reader's Web — here's how you can get involved.

For website owners

Install the free WordPress plugin to publish pages in the Reader's Web format and give your visitors a cleaner, more enjoyable reading experience.

For readers

Install the browser extension to customise the reading theme across all Reader's Web pages — choose your fonts, colours, and layout once and it applies everywhere.
